Ausführliche Beschreibung

Stellt den Zugriff für eine Kamera bereit.

Dazu wird ein Thread mittels runCamera() gestartet, der fortlaufend die Frames der Kamera holt. Als Bibliothek wird die NESTK Bibliothek von Nicolas Burrus verwendet, worin hier die OpenNI Treiber benutzt werden.

Siehe auch:
Nicolas Burrus NESTK Bibliothek http://sourceforge.net/projects/roboticslab/
OpenNI Bibliothek http://www.openni.org/

Beschreibung der Konstruktoren und Destruktoren

CameraModel::CameraModel (  ) 

Default Konstruktur.

Setzt die Kameraparameter der NESTK Bibliothek (kurz NTK) wie folgt:

  • ComputeMapping = true (Gleicht den optischen Abstand zwischen RGB- und monochromen Kamera aus)
  • FilterThresholdDepth = true (Filtert alle Tiefeninformationen die größer als 100cm aus)
  • TriangleMesh (Erzeugt ein Dreiecksnetz der Punktwolke)

bool CameraModel::cameraCalibrationWithLED ( cv::Point3f  startPoint,
int  numX,
int  numY,
RoboticArm *  arm 
)

Findet die extrinischen Parameter der Kamera über ein am Roboterarm angebrachter Leuchtdiode.

Der Roboterarm fährt dazu ein virtuelles Schachbrettmusters ab. Der Anfangspunkt ist der untere linkere Eckpunkt des Schachbrettmusters. Anschließend fährt er spaltenweise das Schachbrettmuster in Abständen von 0.05m ab. Am Ende fährt der Roboterarm in die Ausgangsposition zurück.

Warnung:
Im Bereich vor dem Roboterarm dürfen sich keine Gegenstände sich befinden!
Zu beachten:
In einem verdunkelten Raum funktioniert die Erkennung der Leuchtdiode besser.
Parameter:
startPoint Der Startpunkt in Weltkoordinaten von wo aus der Roboterarm losfährt.
numX Die Gesamtanzahl an horziontalen Bewegungsschritten (Schrittabstand 0.05m).
numY Die Gesamtanzahl an vertikalen Bewegungsschritten (Schrittabstand 0.05m).
arm Der Roboterarm welcher bewegt werden soll.
Rückgabe:
true bei erfolgreicher Kalibrierung, sonst false.
